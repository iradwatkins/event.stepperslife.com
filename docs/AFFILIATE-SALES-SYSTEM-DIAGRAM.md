## Security Architecture

```
┌─────────────────────────────────────────────────────────────────────┐
│                         SECURITY LAYERS                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │ LAYER 1: AUTHENTICATION (NextAuth.js)                        │ │
│  ├───────────────────────────────────────────────────────────────┤ │
│  │ • JWT session tokens (7-day expiry)                           │ │
│  │ • Password hashing (bcrypt, cost: 12)                         │ │
│  │ • Email verification required                                 │ │
│  │ • OAuth providers (Google)                                    │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                      │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │ LAYER 2: AUTHORIZATION (RBAC)                                │ │
│  ├───────────────────────────────────────────────────────────────┤ │
│  │ • Role-based permissions (AFFILIATE role)                     │ │
│  │ • Resource ownership checks                                   │ │
│  │ • Permission validation middleware                            │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                      │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │ LAYER 3: PIN SECURITY                                         │ │
│  ├───────────────────────────────────────────────────────────────┤ │
│  │ • PIN hashing (bcrypt, cost: 12)                              │ │
│  │ • Never stored in plaintext                                   │ │
│  │ • Rate limiting (3 attempts / 15 min)                         │ │
│  │ • Account lockout (5 consecutive failures)                    │ │
│  │ • PIN rotation (forced every 90 days)                         │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                      │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │ LAYER 4: FRAUD DETECTION                                      │ │
│  ├───────────────────────────────────────────────────────────────┤ │
│  │ • Velocity checks (max sales per hour/day)                    │ │
│  │ • IP address tracking & blacklisting                          │ │
│  │ • Self-referral detection                                     │ │
│  │ • Duplicate email detection                                   │ │
│  │ • Admin review queue for flagged transactions                 │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                      │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │ LAYER 5: AUDIT LOGGING                                        │ │
│  ├───────────────────────────────────────────────────────────────┤ │
│  │ • All cash sales logged                                       │ │
│  │ • PIN attempts tracked                                        │ │
│  │ • Payout processing logged                                    │ │
│  │ • Admin actions audited                                       │ │
│  │ • Immutable audit trail (append-only)                         │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```
