# Product Requirements Document (PRD)

This document provides an overview and links to the detailed Product Requirements for the events-stepperslife project.

## Primary Document
The complete and detailed Product Requirements Document has been organized according to the BMAD (Business, Methods, Architecture, Development) methodology:

**Main PRD Document:** [docs/business/product-requirements.md](./business/product-requirements.md)

## BMAD Structure
This project follows the BMAD organizational structure:
- **Business** (B): Product requirements, market analysis, business goals
- **Methods** (M): Development methodologies, processes, workflows
- **Architecture** (A): System design, technical architecture, patterns
- **Development** (D): Implementation details, technology stack, coding standards
